### 자바 눈 피하기 게임
------------------
> 소개   

내려오는 눈을 피하고, 상자를 먹어 점수를 얻는 간단하 자바게임

> 실행 방법
* enter를 누르면 노래와 함께 시작

> 프로그램 소개
* 상자를 먹을 시 점수를 획득, 눈을 맞으면 생명이 1개씩 차감

> 주의 사항
* 시간이 지날 수록 눈과 상자 속도가 더 빨리 내림
* 생명 3개를 모두 소진 후 눈을 맞을 시 게임 종료.
